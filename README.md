CyberLayer is a multi-layered cybersecurity system that detects and prevents cyber attacks such as DDoS, SQL Injection, and Phishing in real time. It combines anomaly detection with targeted attack modules to improve accuracy and reduce false alerts. The system includes secure login with MFA, real-time monitoring, and automated report generation. It was developed using a Hybrid (ASD + Spiral) model, with complete system design, testing (unit, integration, system), and project planning (WBS, COCOMO, risk management). It represents a scalable and practical cybersecurity solution for modern digital environments.
